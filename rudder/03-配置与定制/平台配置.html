初始化平台 ​

bash
# 初始化所有平台
rudder init -u your-name

# 只初始化需要的平台（推荐）
rudder init --claude --cursor --codex -u your-name

更新平台配置 ​

bash
rudder update
1

会自动检测哪些模板文件有新版，并提示是否覆盖。遇到冲突时有三种选择：

  • Overwrite — 用新版本替换
  • Skip — 保留当前版本
  • Create copy — 保存新版本为 .new 文件

提示：可以按 a / s / n 批量应用选择。

版本检测 ​

Rudder CLI 启动时会自动检测：

  • 如果 .rudder/ 存在，会比较 CLI 版本与项目记录版本
  • CLI 版本更新时，提示运行 rudder update
