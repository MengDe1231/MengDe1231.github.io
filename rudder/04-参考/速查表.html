速查表 ​

Rudder 常用命令和操作的快速参考。

CLI 命令 ​

命令说明
rudder --version查看版本
rudder init -u name初始化项目（全平台）
rudder init --claude -u name初始化指定平台
rudder update更新项目配置到最新

任务命令 ​

bash
python3 .rudder/scripts/task.py <command>
1
命令说明快捷写法
create "标题" --slug xxx创建任务create "登录" -s login
start xxx激活任务start login
current --source查看当前任务current -s
finish清除当前任务finish
archive xxx归档任务archive login
list列出活动任务ls
list --mine我的任务ls --mine
list-archive列出已归档ls -a
set-branch xxx branch设置分支set-branch login feat/x
set-base-branch xxx br设置 PR 目标set-base-branch login dev
set-scope xxx scope设置 PR scopeset-scope login auth
add-subtask parent child添加子任务—
remove-subtask parent child移除子任务—

上下文命令 ​

bash

模式说明
无参数完整的 session 上下文
--mode packages列出可用的 Spec 包和层
--mode phase --step 1.3获取某步骤的详细指导

工作流阶段速览 ​

阶段目标关键步骤
Phase 1: Plan搞清楚要做什么create → PRD → JSONL → start
Phase 2: Execute写代码过审查implement → check → fix
Phase 3: Finish沉淀认知收尾verify → update-spec → commit
