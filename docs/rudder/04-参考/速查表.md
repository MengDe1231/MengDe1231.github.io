# 速查表

Rudder 常用命令和操作的快速参考。

## CLI 命令

| 命令 | 说明 |
|------|------|
| `rudder --version` | 查看版本 |
| `rudder init -u name` | 初始化项目（全平台） |
| `rudder init --claude -u name` | 初始化指定平台 |
| `rudder update` | 更新项目配置到最新 |

## 任务命令

```bash
python3 .rudder/scripts/task.py <command>
```

| 命令 | 说明 | 快捷写法 |
|------|------|---------|
| `create "标题" --slug xxx` | 创建任务 | `create "登录" -s login` |
| `start xxx` | 激活任务 | `start login` |
| `current --source` | 查看当前任务 | `current -s` |
| `finish` | 清除当前任务 | `finish` |
| `archive xxx` | 归档任务 | `archive login` |
| `list` | 列出活动任务 | `ls` |
| `list --mine` | 我的任务 | `ls --mine` |
| `list-archive` | 列出已归档 | `ls -a` |
| `set-branch xxx branch` | 设置分支 | `set-branch login feat/x` |
| `set-base-branch xxx br` | 设置 PR 目标 | `set-base-branch login dev` |
| `set-scope xxx scope` | 设置 PR scope | `set-scope login auth` |
| `add-subtask parent child` | 添加子任务 | — |
| `remove-subtask parent child` | 移除子任务 | — |

## 上下文命令

```bash
python3 .rudder/scripts/get_context.py
```

| 模式 | 说明 |
|------|------|
| 无参数 | 完整的 session 上下文 |
| `--mode packages` | 列出可用的 Spec 包和层 |
| `--mode phase --step 1.3` | 获取某步骤的详细指导 |

## 工作流阶段速览

| 阶段 | 目标 | 关键步骤 |
|------|------|---------|
| Phase 1: Plan | 搞清楚要做什么 | create → PRD → JSONL → start |
| Phase 2: Execute | 写代码过审查 | implement → check → fix |
| Phase 3: Finish | 沉淀认知收尾 | verify → update-spec → commit |

## 跳过流程

对 AI 说以下任一话术即可跳过任务流程：

```
skip rudder / 跳过 rudder / 别走流程 / 直接改 / 小修一下 / 先别建任务
```

## 配置项速查

| 配置项 | 默认值 | 说明 |
|--------|-------|------|
| `max_journal_lines` | 2000 | journal 每文件最大行数 |
| `session_auto_commit` | false | 日志/任务归档自动提交 |
| `code_auto_commit` | false | 代码变更自动提交 |
| `session_commit_message` | "chore: record journal" | 会话提交消息模板 |
